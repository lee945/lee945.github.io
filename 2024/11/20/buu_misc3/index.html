[SWPU2019]神奇的二维码

得到一个二维码

扫描得到一串flag，他说“flag is not here”，不对

用010打开发现很多东西，分离一下，很多rar

发现7104.rar不要密码，有一个文件，打开是base编码

解码得到asdfghjkl1234567890

17012.rar不要密码里面还有一个doc，打开这样，也是base

一长串应该也是解码成密码，应该是多次编码，用脚本解码，又得到一个字符串comEON_YOuAreSOSoS0great

import base64
def decode(f):
n = 0;
while True:
try:
f = base64.b64decode(f)
n += 1
except:
print(‘[+]Base64共decode了{0}次，最终解码结果如下:’.format(n))
print(str(f,’utf-8’))
break
if name == ‘main’:
f = open(‘./b64.txt’,’r’).read()
decode(f)

comEON_YOuAreSOSoS0great是18394.rar的密码，里面有个mp3
