js入门

介绍

JavaScript 是一种轻量级的解释型或即时编译型的编程语言，主要用于网页开发，为网页添加交互功能。它没有严格的语法限制，可以很容易地学习和使用。

JavaScript 的特点

  • 解释型语言：不需要编译，由 JavaScript 引擎直接解释执行
  • 动态类型：变量类型在运行时确定
  • 基于原型：使用原型继承而非类继承
  • 多范式：支持面向对象、函数式和命令式编程风格
  • 单线程：使用事件循环和回调机制处理并发

比较运算符

  • =，如：1与”1”
  • == 等于（类型不一样，值一样，也会判断为true）
  • === 绝对等于（类型一样，值一样，结果为true）
  • 尽量不要使用 == 比较
  • NaN === NaN，这个与所有的数值都不相等，包括自己。
