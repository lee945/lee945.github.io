CSS的基本概念和作用

CSS又称层叠样式表，是一种样式表语言，用于描述HTML或XML文档的外观和格式。

css用处

在html中引入css

一般有三种引入方法：内联样式、内部样式表、外部样式表。

基础的html结构：

内联样式

内联样式就是在HTML元素直接使用style属性。

虽然简单，但是不易于维护，不建议大量使用

内部样式表

将CSS代码放在HTML文档的 <head> 部分中的 <style> 标签内。

适用于单个页面的样式定义，但样式不能跨页面复用。

外部样式表

将CSS代码保存在一个单独的 .css 文件中，然后通过 <link> 标签在HTML文档中引入。

它允许多个页面之间共享样式，维护和更新更容易。（推荐）

styles.css

CSS语法

CSS的语法规则相对简单，由选择器、属性和值组成。

选择器

选择器用于指定要样式化的HTML元素。简单来说，选择器的作用就是“指出”我们要对网页上的哪些部分进行操作。

常见的选择器

元素选择器

根据元素的名称来选择HTML元素。

类选择器

通过元素的class属性值来选择元素。类选择器前面有一个点（.）。

ID选择器

通过元素的id属性值来选择元素。ID选择器前面有一个井号（#）。

属性选择器

根据元素的属性及其值来选择元素。它们以方括号（[]）表示。

通配选择器

伪类选择器

用于选择处于特定状态的元素。它们以冒号（:）开头。

伪元素选择器

用于选择元素的特定部分。它们以双冒号（::）开头，但有时也可以使用单冒号（:）。

组合选择器

用于选择满足多个条件的元素

后代选择器（空格）

子选择器（>）

相邻兄弟选择器（+）

通用兄弟选择器（~）

群组选择器（，）

分层选择器

允许你根据文档结构进行选择

选择器优先级

优先级就是，当多个规则应用于同一个元素时，怎么确定应用哪些样式规则。

浏览器会根据以下规则来确定优先级：

!important > 内联样式 > ID选择器 > 类选择器、属性选择器和伪类选择器 > 元素选择器和伪元素选择器 > 通配符选择器（*），关系选择器（如后代选择器、子选择器、相邻兄弟选择器等），以及从父元素继承的样

权重级别 规则
1 给文档中的元素设置style属性
2 ID选择器
3 类选择器、属性选择器和伪类选择器
4 标签（元素）选择器和伪元素选择器
5 通配选择器和继承样式

强调：

优先级高的样式会覆盖优先级低的样式。

如果两个选择器具有相同的优先级，则后面的样式会覆盖前面的样式。

使用!important声明可以覆盖所有其他优先级，但不推荐使用，因为后续调试困难。

属性选择器的使用

属性选择器可以根据元素的属性及其值来选择元素。

伪类选择器的使用

选择器 类别 描述 示例
:link 链接伪类 选择未被访问的链接 a:link { color: blue; }
:visited 链接伪类 选择已被访问的链接 a:visited { color: purple; }
:hover 用户交互伪类 选择鼠标悬停的元素 button:hover { background: #eee; }
:active 用户交互伪类 选择被激活的元素（如鼠标按下时的状态） a:active { font-weight: bold; }
:focus 用户交互伪类 选择获得焦点的元素（如输入框、按钮） input:focus { border-color: red; }
:first-child 结构伪类 选择其父元素的第一个子元素 li:first-child { color: green; }
:last-child 结构伪类 选择其父元素的最后一个子元素 div p:last-child { margin-bottom: 0; }
:nth-child(n) 结构伪类 选择其父元素的第n个子元素（支持数字/表达式） tr:nth-child(2n) { background: #f5f5f5; }
:nth-last-child(n) 结构伪类 选择其父元素的倒数第n个子元素 li:nth-last-child(2) { font-style: italic; }
:not(selector) 否定伪类 选择不符合指定选择器的元素 div:not(.hidden) { display: block; }

属性

字体样式

属性 描述
font-size: 字体大小，可以使用像素值（px）、百分比（%）、em单位等
font-family: 字体种类，可以设置多个备选字体，用逗号隔开
font-weight: 字体粗细，一般有normal（普通）、bold（加粗）两种,也可用数值表示（100、200、300…）
font-style: 字体风格，可以设置italic（斜体）或normal（正体）

文本样式

属性 描述
color: 字体颜色，可以使用十六进制、RGB、RGBA、HSL、HSLA等。(“ A “透明度)
text-align: 文本水平对齐方式，可以设置left、center、right等
line-height: 行高，用于设置行与行之间的垂直距离
text-decoration: 文本修饰（如下划线、删除线等），可以设置underline、line-through等
text-transform: 文本转换（如大写、小写等），可以设置uppercase、lowercase等

盒子模型样式

属性 描述
width: 盒子宽度，可以使用像素值（px）、百分比（%）等
height: 盒子高度，与宽度类似
margin: 盒子外边距，可以设置上、右、下、左四个方向的外边距值，也可以只设置其中的某几个
padding: 盒子内边距，与外边距类似
border: 盒子边框，可以设置边框类型、粗细、颜色等
box-sizing: 盒子模型的计算方式，可以设置content-box（默认）或border-box,如果设置为border-box，width属性将包含元素的内容、内边距和边框

背景样式

属性 描述
background-color: 背景颜色，与color属性类似
background-image: 背景图像，可以设置图片的URL地址
background-repeat: 背景图像平铺方式，可以设置repeat（平铺）、no-repeat（不平铺）等
background-position: 背景图像位置，可以设置像素值（px）或百分比（%）

布局属性

属性 描述 示例
display 设置元素的显示类型 block
float 设置元素的浮动位置 left
position 设置元素的定位方式 absolute
top, right, bottom, left 设置定位元素的位置 top: 20px; left: 30px;
flex-direction 在Flexbox布局中设置主轴的方向 row

动画和过渡属性

属性 描述 示例
transition 设置元素的过渡效果 transition: all 0.3s ease;
animation 设置元素的动画效果 animation: slide 2s infinite;

其他属性

属性 描述 示例
opacity 设置元素的透明度 opacity:0.5;
cursor 设置鼠标光标样式 cursor:pointer;
overflow 设置内容溢出时的行为 overflow:hidden;
